PseudoClases/Elementos

Pseudo Clases

Las pseudoclases, junto con los pseudoelementos, nos permiten aplicar un estilo a un elemento. No sólo en relación con el contenido del documento, sino también en relación a factores externos. Como el historial del navegador(:visited), el estado de su contenido(:checked) o la posición del mouse (:hover)

Hover

Se activa cuando el user pasa el mouse sobre un elemento

a { background-color: #e11d48; }

Pasame el mouse por encima

a:hover{ background-color: #8c54fb; }

Pasame el mouse por arriba

Active | Link | Visited

Active

La pseudo clase :active se usa comunmente en los elementos <a> y <button>

a:active { color: #e11d48; }

nucba.com.ar
visited

La pseudo clase :visited representa los enlaces que el usuario ya ha visitado.

a:visited { color: purple; }

nucba.com.ar
link

La pseudo clase :link representa un elemento que aún no se ha visitado.

a:link {color: salmon; }

nucba.com.ar.co

Focus

La pseudo clase :focus reprensenta un elemento que ha recibido foco. Se activa cuando el user hace click, toca un elemento o lo selecciona con la tecla Tab del teclado.

input:focus{ background: cornflowerblue; color: white }

First child

La pseudo clase :first-child representa el primer elemento entre un grupo de elementos hermanos

Last child

La pseudo-clase :last-child representa el último elemento entre un grupo de elementos hermanos.

First of type

La pseudo-clase :first-of-type representa el primer elemento de su tipo entre un grupo de elementos hermanos

Nth child

La pseudo-clase :nth-child() coincide con uno o más elementos en función de su posición entre un grupo de hermanos.

Tenemos algunos valores para usar

Odd

Representa a los elementos cuya posición es impar

Even

Representa los elementos cuya posición es par

Con Numero

Representa el elemento que le digamos

Numero n

Representa cada cuantos elementos seleccionamos

Numero n + Numero

Representa a partir de que elemento seleccionamos y cada cuantos.

PseudoElementos

Al igual que las pseudo-clases, los pseudo-elementos se añaden a los elementos, pero la diferencia está en que no describen un estado en especial, si no que se permite añadir estilos a una parte concreta del documento.

First letter

El pseudo elemento ::first-letter aplica estilos a la primera letra de un párrafo.

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Commodi aspernatur officiis explicabo consectetur quas molestias quia voluptates eum alias? Rem!

First Line

El pseudo-elemento ::first-line aplica estilos a la primera linea de un párrafo.

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Error molestias aperiam ad fugiat quod vitae aliquid officia tempore unde nisi.

Before

El pseudo-elemento::before crea un pseudo-elemento antes del elemento. Se usa normalmente para agregar contenido estético a un elemento.

Hola. Tengo un before

After

El pseudo-elemento ::after crea un pseudo-elemento después del elemento. Se usa normalmente para agregar contenido estético a un elemento.

Hola tengo un after

Placeholder

El pseudo-elemento ::placeholder representa al placeholder de un elemento <input>
